# gift

Github + NFT = gift

根据用户的 star 和 followers 数量，生成头像 NFT，同时必须满足至少注册了 1 年。

你的 star 数跟 follower 数，代表了你的实力，但是实力并不能决定你一定会比别人得到更好的 NFT，只能让你机会更大，所以，运气很重要。

0 ~ 2: 初级
3 ~ 5: 中级
6 ~ 8: 高级
9: 特级

Head: 0 ~ 9
Mouth: 0 ~ 9
